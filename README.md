# Customer Churn Analysis

## Project Overview

Customer Churn Analysis is a Machine Learning and Data Analytics project that helps identify the factors responsible for customer subscription cancellations.

The project analyzes customer behavior, engagement levels, activity patterns, and subscription details to determine which customers are likely to churn. It also provides insights that businesses can use to improve customer retention.

---

# Problem Statement

Businesses often lose customers due to low engagement, poor user experience, or inactivity. Identifying churn patterns early helps companies take preventive actions and improve customer satisfaction.

This project focuses on:
- Analyzing customer activity
- Understanding behavioral trends
- Detecting churn patterns
- Predicting customer churn using Machine Learning

# Dataset Columns

| Column Name | Description |
|---|---|
| CustomerID | Unique customer ID |
| Age | Customer age |
| Gender | Male/Female |
| TenureMonths | Subscription duration |
| SubscriptionType | Basic/Standard/Premium |
| MonthlyCharges | Monthly subscription fee |
| TotalCharges | Total amount paid |
| SupportTickets | Number of complaints/support requests |
| LoginFrequency | User login activity |
| AvgSessionTime | Average usage time |
| LastActivityDaysAgo | Inactive days |
| PaymentMethod | Payment type |
| ContractType | Monthly/Quarterly/Yearly |
| Churn | Customer left or not |

## Random Forest Classifier

The model predicts whether a customer is likely to churn based on customer activity and behavioral patterns.
